Lightweight zeroconf ESLint plugin for Atom.

* ESLint is already highly configurable through `.eslintrc` files.
* Same behavior as: `$ cd /path/to/file && eslint src.js`.
* It's fast !

## Installation

* `apm install fast-eslint`
